# Udemy-clone-project
Udemy Clone (HTML & CSS)

This project is a simple clone of the Udemy website, built using only HTML and basic CSS. It focuses on recreating the layout and design of an online learning platform.

The project includes a homepage with course sections, navigation bar, and basic styling to resemble a real-world e-learning website. It helped me understand core front-end concepts like layout design, positioning, and styling.

🚀 Features
Responsive homepage layout (basic)
Navigation bar
Course cards section
Clean and simple UI design
Built using pure HTML & CSS (no frameworks)
🛠️ Technologies Used
HTML5
CSS3
📚 What I Learned
Structuring web pages using HTML
Styling using CSS
Flexbox basics
Layout design techniques
